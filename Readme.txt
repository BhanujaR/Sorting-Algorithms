->A folder containing insertionsort.py, mergesort.py, and quicksort.py should be opened in an enironment like visual studio code.
  insertionsort.py should be executed first followed by mergesort.py and quicksort.py.
commands to execute python files
--> python insertionsort.py
--> python mergesort.py
--> python quicksort.py
->Executing insertionsort.py generates sample data files (arr20.txt, arr100.txt, arr1000.txt, and arr4000.txt ) 
  and output files(arrIS_O_20.txt, arrIS_O_100.txt, arrIS_O_1000.txt, and arrIS_O_4000.txt).
->Executing mergesort.py generates sample data files (arrMR_O_20.txt, arrMR_O_100.txt, arrMR_O_1000.txt, and arrMR_O_4000.txt)
->Executing quicksort.py generates sample data files (arrQK_O_20.txt, arrQK_O_100.txt, arrQK_O_1000.txt, and arrQK_O_4000.txt)
